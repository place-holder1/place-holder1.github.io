Scrap Metal Studios: Project Dyscharged

Dialogue System(s)

As I'm interested in narrative designing, I am always fond of trying to create dialogue systems.

In Unreal Engine 5, I have made two different versions of dialogue systems: Simple & Dynamic. A simple dialogue system involves interacting NPCs within proximity. A dynamic dialogue system I have in mind includes having different conversations and choices every interact prompt based on stats. 

           Simple Dialogue System                     Dynamic Dialogue System


Each conversation will have its own unique ID in the dialogue tree. Once a choice has been made, it will then draw a reference to the conversation ID. I designed this for Scrap Metal Studio programmers to gather reference of what they need to create.

Data Visualization & Programming

An introduction to learning how to examine and create data visualizations using Excel and P5.js.

For my final project, I worked with Aaron White to create a data visualization of measurements of the human body. I was the programmer and debugger for this project. The only known issue with the code is the dropdown positioning due to p5.js and JavaScript conflicting with each other. However, the professor was impressed at the complexity so he let it slide. Computer Graphics Programming: Asteroids

Using C++ with SFML library with Professor Kesselman's physics library, my goal is to recreate Asteroids in Visual Studio. As someone who is familar with C++, I really like to tackle this amazing challenge set to me.

On top of programming, I plan to design my own version of Asteroids. The design plan is for the player to destroy as many asteroids as they can until they collide with an asteroid. The player has one life, and the difficulty progresses adaptively based on how long each asteroid is destroyed.

This is currently my in-progress work.
